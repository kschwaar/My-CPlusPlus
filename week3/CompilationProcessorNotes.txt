Comilation model C uses:

Preprocessor:
	Removes comments
	Interprets Preprocessor directives: #
	
Compiler:
	Source code is converted to assembly language
	
Assembly:
	Assembly language to machine code
	
Linker:
	Links any libraries or functions defined in other files.
